SequelizeAcl

creates a SequelizeAcl instance with all authorization.

new SequelizeAcl(seql: Sequelize, options: Object)
Parameters
seql (Sequelize) A Sequelize instance
options (Object) custom config object
Name Description
options.prefix string (default 'acl_') prefix for acl tables in database
options.primaryKey string (default 'id') primary key for acl tables
options.timestamps bool (default false) add timestamps to tables
options.paranoid bool (default false) soft deletes of items
options.sync bool (default true) auto create tables, set this to false if your are using migrations
options.debug bool (default false) debug mode, output logs
options.UserModel Model (default null) Model used for User, make sure you pass model and not name(string)
options.userPk string (default 'id') primary key used for User Model
options.safeAclDeletes bool (default true) role or permissions can't be deleted as long as they are associated with any other data. To remove you must break all other associations (upcoming)
options.userCache bool (default true) roles of user will be cached, this will allow faster permission resolution and less database connections
options.userCacheTtl int (default 60) time for which roles of user will be cached (in seconds), this number should be inversely proportional to your user traffic.

▸ findPermissions(args)

Find Permissions. returns all permissions if no arg passed.

findPermissions(args: Object): Promise<Array>
Parameters
args (Object = {}) search object
Name Description
args.name name name of permissions to find
args.resource string name of resource eg. blog, post
args.action string nae of action eg. view, edit
args.search bool if set to true will perform match on the name, resource and action, otherwise exact match.
Returns
Promise<Array>: array of permissions as model instances
Example
acl.findPermissions().then((perms) =>{});

 acl.findPermissions({name : 'admin'}).then((perms) =>{});

 acl.findPermissions({resource : 'blog', action : 'view'}).then((perms) =>{});

 acl.findPermissions({action : 'view', search : true}).then((perms) =>{});

 // perms will be array of permissions model objects.
